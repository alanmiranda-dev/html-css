Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase, temos um termo em destaque usando a tag <strong>

Nesta frase, temos um termo em itálico usando a tag (nao semantica) <i>

Nesta frase temos um termo em ênfase usando a tag EM (semantica)

Textos marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande (não semantico) e um texto pequeno (semantico) neste parágrafo

Texto deletado

Podeos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Texto Sobrescrito

Para inserir coisas do tipo x20+3

Texto SUBescrito
